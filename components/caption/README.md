# caption

## Tags

- `<(label|span|div|*) data-shape=caption>`

## Styled attributes

- `... data-shape=(textinput|checkbox|radiobutton|<component_type>)>`<br>
to set the vertical spacing consistent with the specified shape (e.g. in form field rows or table rows)
