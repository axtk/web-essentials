# tooltip

## Tags

- `<(div|*) data-shape=tooltip>`

## Styled attributes

- `... data-tail=(<orientation>)>`

---

## Example

```
<div data-shape=tooltip data-tail=bottom-left>
    Lorem ipsum auctor neque nec sapien ipsum, porta auctor quis.
</div>
```
